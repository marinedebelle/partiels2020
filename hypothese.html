Hypothèses

Hypothèse 1 - web

Le but ici est de réaliser une interface web où l'utilisateur pourra, grâce à son téléphone, jauger ses émotions de 0 à 10 en bougeant des curseur. Grâce à un serveur, ces données seront transférées à l'installation Arduino qui allumera et changera l'intensité lumineuse de chaque LED selon l'intensité de l'émotion ressentie.

Outils
  • Interface web (UI / UX) sur mobile
  • Serveur (node.js)
  • Arduino (C + matériel)
  • Bandeau LED RGB
Besoins techniques
  • - Hébergement (OVH)
  • HTML / Css / Js / Node.js
  • RGB LED strip
  • Arduino Uno
  • 10k Ohm Resistors
  • Logic Level N-channel MOSFETs
  • Breadboard
  • Hookup wires
  • 12v Power Supply

Hypothèse 2 - boutons

Le but ici est de réaliser une installation Arduino où l'utilisateur pourra, grâce à des boutons, jauger ses émotions de 0 à 10 en appuyant une ou plusieurs fois sur les boutons correspondant à celles-ci (exemple : le bouton rouge représente la colère et la personne appuyant 8 fois sur ce bouton ressentira très fortement cette émotion). Grâce au code et à l'installation Arduino, la lumière LED s'allumera si la personne ressent l'émotion (si elle a appuyé sur le bouton) et son intensité changera selon le nombre de pressions faites sur le bouton.

Outils
  • Arduino (C + matériel)
  • Bandeau LED RGB
Besoins techniques
  • RGB LED strip
  • Arduino Uno
  • 10k Ohm Resistors
  • Logic Level N-channel MOSFETs
  • Breadboard
  • Hookup wires
  • 12v Power Supply
  • boutons

Hypothèse 3 - lunettes

Le but ici est de réaliser une interface web où l'utilisateur pourra, grâce à son téléphone, jauger ses émotions de 0 à 10 en bougeant des curseur. Grâce à un serveur, ces données seront transférées à l'installation Arduino qui allumera et changera l'intensité lumineuse de chaque LED selon l'intensité de l'émotion ressentie. Les LED seront placées à l'intérieur de lunettes hermétiques à la lumière extérieure. Petit "+" → 2 émotions peuvent avoir la même couleur mais pas le même rythme (la colère sera représentée par un rouge clignotant tandis que l'amour par un rouge doux et lent).

Outils
  • Interface web (UI / UX) sur mobile
  • Serveur (node.js)
  • Arduino (C + matériel)
  • LEDs de différentes couleurs
  • Imprimante 3D
Besoins techniques
  • Hébergement (OVH)
  • HTML / Css / Js / Node.js
  • RGB LED strip
  • Arduino Uno
  • 10k Ohm Resistors
  • Logic Level N-channel MOSFETs
  • Breadboard
  • Hookup wires
  • 12v Power Supply

Hypothèse 4 - coronavirus

Le but ici est de réaliser une interface web où l'utilisateur pourra, grâce à son téléphone, jauger ses émotions de 0 à 10 en bougeant des curseur. Après avoir entré ces données, il aura a valider celles-ci et un autre écran apparaitra. Dans celui-ci les couleurs apparaitront comme des tâches de lumière : fortement pour les émotion ressenties vivement et moins intenses pour celles ressenties faiblement.

Petits "+"

  • L'utilisateur pourra entrer son nom dans le site pour que celui-ci s'affiche lors de résultats
  • L'utilisateur pourra consulter son historique d'émotions
  • L'utilisateur pourra consulter les émotions collectives (écran où toutes les émotions des gens ayant participé à l’expérience apparaitront)
Outils
  • Interface web (UI / UX) sur mobile et ordinateur
Besoins techniques
  • HTML / Css / Js
  • Storage navigator
